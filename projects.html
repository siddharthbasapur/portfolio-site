SafeMed Automatic Pill Organizer
Arduino • Embedded Systems • Accessibility • Product Design

SafeMed is a fully functioning automatic pill organizer designed to assist seniors (65+) in managing their daily medication schedules safely and independently. The system addresses a critical real-world problem—missed or incorrect medication dosing—by automating the dispensing process.

The device uses an embedded control system with a keypad interface to allow users to interact with the system, set schedules, and retrieve medication. Internally, the system integrates Arduino-based control logic, power systems, and a mechanical dispensing mechanism.

I designed and built the entire prototype, including circuit integration, enclosure construction, and system logic. A major focus of the project was usability—ensuring the system is simple, reliable, and accessible for elderly users.

Arduino Embedded Systems Circuit Design Hardware Prototyping User-Centered Design
  • Automated medication dispensing system
  • Keypad-based user interface
  • Integrated electronics and control logic
  • Physical prototype enclosure design
  • Focused on accessibility and real-world usability
Drone Fertilizer Attachment Prototype
SkillsUSA • Arduino • Agricultural Engineering

This project explored the use of drone technology to improve agricultural efficiency, specifically in spreading fertilizer across uneven or hard-to-reach terrain.

I developed a working prototype attachment using Arduino circuitry, wiring systems, and a structural model to simulate how fertilizer distribution could be automated through aerial systems.

The project required combining electronics, mechanical design, and systems thinking to translate a real-world agricultural problem into a functional engineering solution.

Arduino Electronics Prototyping Engineering Design
  • Designed drone-based fertilizer system concept
  • Integrated electronics and circuit components
  • Built physical prototype structure
  • Applied engineering design to real-world agriculture problem
