Drone Fertilizer Attachment Prototype

SkillsUSA • Arduino • Hardware Prototyping

Designed and built a prototype drone attachment to help farmers spread fertilizer over rough terrain. This project focused on solving real-world agricultural challenges using low-cost engineering solutions.

The system integrates Arduino components, wiring, and a custom-built physical structure to demonstrate how automated fertilizer distribution could work in difficult environments.

Arduino Circuit Design Prototyping Engineering Design
  • Built a working physical prototype
  • Integrated electronics and power system
  • Designed solution for real agricultural problem
  • Presented in SkillsUSA competition

Automatic Pill Organizer

Arduino • Embedded Systems • Accessibility

Built a fully functioning automatic pill organizer designed to help seniors (65+) manage medication schedules safely and independently.

The system uses a keypad interface and internal control logic to dispense pills at scheduled times, focusing on accessibility, ease of use, and reliability.

Arduino Embedded Systems Circuit Design Product Design
  • Automated medication dispensing system
  • Keypad-based user interface
  • Designed specifically for elderly users
  • Built physical enclosure and electronics system
