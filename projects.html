SafeMed Automatic Pill Organizer
Arduino • Embedded Systems • Accessibility Design

SafeMed is a fully functioning automatic pill organizer designed to help seniors (65+) manage medication schedules safely and independently. The system automates pill dispensing using a keypad interface and embedded control system.

I designed and built the full prototype, integrating electronics, control logic, and a physical enclosure. The project focused on accessibility, reliability, and real-world usability.

  • Keypad interface for user control
  • Designed for elderly accessibility
